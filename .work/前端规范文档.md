## 1. 设计理念与愿景

### 1.1 产品定位
SwallowNote 是一款面向**开发者和技术写作者**的跨平台 Markdown 编辑器，主打**高性能**、**本地优先**和**AI 智能辅助**。

### 1.2 设计愿景
打造一款**专业且克制**的编辑工具——界面简洁但功能完整，视觉上借鉴 VS Code 的效率美学，但更现代、更轻盈。

### 1.3 设计原则

| 原则 | 说明 |
|------|------|
| **效率优先** | 信息密度高，操作路径短，键盘驱动 |
| **克制美学** | 减少视觉噪音，让内容成为焦点 |
| **可预测性** | 遵循平台惯例，保持操作一致性 |
| **性能至上** | 虚拟滚动、按需加载、零卡顿 |

## 2. 设计语言

### 2.1 美学方向

**风格**: Industrial-Minimalist（工业极简）
- 参考：VS Code + Linear + Raycast
- 特点：深色为主、强调功能性、微妙的层级感

**色彩策略**:
- 主色：饱和度适中的蓝 (`#4A90D9`)，在深色背景下不刺眼
- 强调色：琥珀色 (`#F5A623`) 用于提醒和高亮
- 功能色：绿色（成功）、红色（错误/删除）、黄色（警告）

  name: string;

**标签样式**:

- Hover：背景 `--bg-hover`
- Dirty：显示圆点 `●`
- Pinned：显示图钉图标

**交互**:
- 单击：切换标签
- 中键/双击：关闭标签
- 拖拽：调整顺序
- 右键：上下文菜单
