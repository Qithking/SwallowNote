创建一个 pc 端 markdown笔记软件，支持跨平台: windows, mac, linux
   1.布局：资源管理区、编辑区，底部状态栏区，其中资源管理区和编辑区是左右布局，状态栏区在资源管理区和编辑区下面，高度固定
   2.资源管理区：是纵向布局，依次是:
       工具栏：
         左边：关闭、最小化、最大化图标     
         右边：主题下拉选择，设置    
       tab 切换区：资源管理器、搜索、同步
       内容区：
        资源管理器：
            工具栏：打开文件夹，新建文件，新建文件夹，刷新，折叠/张开
            文件展示区：使用树形组件加载文件夹，支持展开/折叠文件夹，支持文件/文件夹右键菜单（新建、重命名、删除、复制路径等），支持拖拽排序和移动文件，图标区分文件类型（Markdown、图片、其他），显示 Git 状态标识（新增、修改、删除、未跟踪）
               
        搜索：
            搜索框：支持全局内容搜索，正则表达式支持，文件过滤，搜索结果高亮，批量替换
            结果区：搜索结果以列表的形式显示，显示内容包含文件名，和部分内容，高亮显示搜索关键词，可以参考 vscode 的搜索            
        同步：
            工具栏：同步按钮：同步下面内容区选择的 git 仓库，如果选择仓库为空则表示全部同步，同步完成后刷新下面内容区
            内容区：显示待提交的 git仓库，支持多选                
        
   3.编辑区：是纵向布局，依次是:
       tab栏：展示所有打开的文件，文件有编辑未保存时，显示一个红色小圆点，保存完成后显示绿色小圆点，支持拖拽排序，tab 项右键支持关闭当前 tab 项，关闭其他 tab 项，关闭全部 tab 项，支持 tab 项快捷键导航（Ctrl+Tab 切换）
       工具栏：
            左边：显示当前文件的路径、大小、修改时间
            右边：目录、源码、git、ai、setting，展开/关闭图标
       内容区：左右布局
            左边：文件内容显示区
            右边：工具栏操作扩展区，例如：git、ai、setting      
   4.状态栏区：横向布局
        左边：应用名称、版本号、作者、github 链接
        右边：当前文件字数、git状态、修改时间
